JQuery API

Nếu bạn đang cập nhật lên phiên bản jQuery mới hơn, hãy nhớ đọc ghi chú phát hành được công bố trên blog của chúng tôi. Nếu bạn đến từ phiên bản 1.9 trước, bạn cũng nên xem Hướng dẫn nâng cấp 1.9. Lưu ý rằng đây là tài liệu API cho lõi jQuery. Các dự án khác có tài liệu API ở các vị trí khác:


  • jQuery UI API docs
  • jQuery Mobile API docs
  • QUnit API docs

.add()

Tạo một đối tượng jQuery mới với các phần tử được thêm vào tập hợp các phần tử phù hợp.

.find()

Lấy hậu duệ của từng phần tử trong tập hợp các phần tử phù hợp hiện tại, được lọc bởi bộ chọn, đối tượng jQuery hoặc phần tử.

.hide()

Ẩn các yếu tố phù hợp.

.toggle()

Liên kết hai hoặc nhiều trình xử lý với các phần tử phù hợp, để được thực hiện trên các nhấp chuột thay thế.

.show()

Hiển thị các yếu tố phù hợp.

.animate()

Thực hiện một hình ảnh động tùy chỉnh của một tập các thuộc tính CSS.

.fadeIn()

Hiển thị các yếu tố phù hợp bằng cách làm mờ chúng thành mờ.

.fadeOut()

Ẩn các yếu tố phù hợp bằng cách làm mờ chúng trong suốt.

.slideUp()

Ẩn các yếu tố phù hợp với một chuyển động trượt.

.slideDown()

Hiển thị các yếu tố phù hợp với một chuyển động trượt.

.html()

Lấy nội dung HTML của phần tử đầu tiên trong tập hợp các phần tử phù hợp hoặc đặt nội dung HTML của mọi phần tử phù hợp.

.append()

Chèn nội dung, được chỉ định bởi tham số, vào cuối mỗi phần tử trong tập hợp các phần tử phù hợp.
